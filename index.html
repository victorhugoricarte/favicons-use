⭐ Utilização de Favicons

Pequeno ícone que identifica um site nas guias abertas no navegador, chamado de favicon ou shortcut icon (Favorite icon)

→ Adicionado dentro do conteúdo do <head> pelo elemento <link>

→ A tag <link> não tem fechamento, para ser responsável por fazer a ligação entre referências de hipertexto, deve conter os seguintes atributos e parâmetros:

  1. rel = "shortcut icon"
  2. href = "caminho/para/o/ícone"
  3. type = "image/svg+xml"
    • Aqui o atributo type é considerado de modo diferente do habitual, para a utilização do mecanismo MIME types;
    • Esse mecanismo é considerado uma das maneiras de transmitir informações do tipo de documento em questão, ou seja, a extensão do arquivo;
    • Nesse caso, o mecanismo está especificando que é uma imagem de extensão .svg
